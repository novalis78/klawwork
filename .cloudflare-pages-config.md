# Cloudflare Pages Build Configuration

After monorepo restructure, update your Cloudflare Pages settings:

## Build Settings

**Root directory:** `website`

**Build command:** `npm run build`

**Build output directory:** `dist`

**Node version:** `18` or higher

## How to Update

1. Go to https://dash.cloudflare.com
2. Navigate to Pages → keywork
3. Go to Settings → Builds & deployments
4. Update "Root directory (advanced)" to: `website`
5. Save and trigger a retry of the latest deployment

## Or use wrangler pages:

```bash
cd website
npx wrangler pages deploy dist --project-name=keywork
```
